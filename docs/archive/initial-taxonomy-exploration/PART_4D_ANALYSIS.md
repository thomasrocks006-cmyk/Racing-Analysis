### 15.1 Last Start Race Replay Review (Visual Performance Assessment)

**Importance**: 9/10 (Visual evidence of last performance)

**Prediction Power**: 10-15% (Particularly for identifying excuses or hidden strengths)

**Data Availability**: Hard (Requires video analysis)

**Acquisition Strategy**:
- **Primary**: Racing.com race replays (video)
- **Method**: Human observation OR Future AI vision analysis
- **Extract**: Running patterns, checks, wide runs, finishing effort

**Source Specifics**:
```
Race Replay Data Sources:

1. Racing.com Race Replays:
   - URL: https://www.racing.com/replays/[race-id]
   - Format: Video (MP4 streaming)
   - Quality: HIGH (official broadcast footage)
   - Availability: All races (free public access)
   - Method: Video streaming (requires viewing)

2. Racing.com Replay Analysis Articles:
   - URL: https://www.racing.com/news/replay-analysis
   - Format: Written analysis of key races
   - Quality: HIGH (expert analysts describe key moments)
   - Method: Web scraping (text extraction, easier than video)

3. Racing NSW/VIC Official Replays:
   - URL: https://www.racingnsw.com.au / https://www.racing.com/vic
   - Format: Video replays (official)
   - Quality: HIGH
   - Method: Video streaming

Race Replay Analysis Categories:

1. Excuses (Legitimate Bad Luck):
   - Visual Evidence: Checked heavily (interfered with, lost momentum)
   - Visual Evidence: Raced wide throughout (no cover, extra distance)
   - Visual Evidence: Blocked for clear run (jockey had to check, couldn't get through)
   - Visual Evidence: Stumbled at start (lost 3-5 lengths, impossible to recover)
   - Visual Evidence: Difficult barrier (wide, forced wide immediately)

   Example Excuse (Checked Heavily):
   ```
   Last Start Replay Observation:
   - Horse X approaching straight, 3rd position, traveling well
   - 200m to go: Runner in front shifted out suddenly (interference)
   - Horse X checked heavily (jockey had to pull hard to avoid collision)
   - Horse X lost all momentum, fell back to 6th
   - Assessment: LEGITIMATE EXCUSE (bad luck, not poor performance)
   - Prediction Impact: FORGIVE POOR RESULT COMPLETELY (+15-20% boost, ignore last start result)
   ```

2. Hidden Strengths (Better Than Result Suggests):
   - Visual Evidence: Raced wide but still finished strong (extra ground covered, very impressive)
   - Visual Evidence: No clear run but tried to push through (good effort, unlucky)
   - Visual Evidence: Sat three-wide entire race but still placed (very tough run, elite effort)
   - Visual Evidence: Closing fast at finish (strong finishing sprint, needed extra 50-100m)

   Example Hidden Strength (Closing Fast):
   ```
   Last Start Replay Observation:
   - Horse Y finished 4th (moderate result on paper)
   - Replay shows: Horse Y settled last, 10 lengths off leader
   - Final 400m: Horse Y unleashed huge finishing burst
   - Passed 6 horses in straight, closing very fast at finish line
   - Assessment: MUCH BETTER THAN RESULT SUGGESTS (needed extra distance)
   - Prediction Impact: +10-15% boost (strong finishing effort, step up to 1600m today ideal)
   ```

3. Exposed Weaknesses (Worse Than Result Suggests):
   - Visual Evidence: Won but had easy time (soft lead, no pressure, flattered)
   - Visual Evidence: Struggled to hold on (caught on line, nearly beaten)
   - Visual Evidence: Weak finishing effort (faded badly final 100m, fitness concern)
   - Visual Evidence: Rider working hard entire race (horse not traveling, laboring)

   Example Exposed Weakness (Soft Lead Flattered):
   ```
   Last Start Replay Observation:
   - Horse Z won last start (looks good on paper)
   - Replay shows: Horse Z led uncontested (no other leaders, soft pace)
   - Sat in front with huge break (6 lengths clear, easy time)
   - Rivals closed fast final 200m (Horse Z barely held on)
   - Assessment: RESULT FLATTERED (easy lead, barely held on when challenged)
   - Prediction Impact: -10-15% penalty (result misleading, fast pace today will expose)
   ```

4. Running Pattern Quality:
   - Visual Evidence: Traveled smoothly entire race (good action, effortless)
   - Visual Evidence: Changed gears quickly (acceleration impressive)
   - Visual Evidence: Laboring action (poor action, struggling)
   - Visual Evidence: Rider niggling (jockey working hard, horse not responsive)

   Example Quality Pattern (Traveled Smoothly):
   ```
   Last Start Replay Observation:
   - Horse A traveled smoothly mid-pack (relaxed, good action)
   - Jockey sitting quietly (not pushing, horse traveling within itself)
   - Final 200m: Jockey asked horse to quicken (immediate response, quick acceleration)
   - Strong finishing effort (passed 3 rivals easily)
   - Assessment: HIGH QUALITY PERFORMANCE (traveled well, good acceleration)
   - Prediction Impact: +8-12% boost (quality run, good signs)

### 15.2 Running Style & Pattern Observation (Action Quality Assessment)

**Importance**: 7/10 (Running action quality)

**Prediction Power**: 5-8% (Good action indicates fitness)

**Data Availability**: Hard (Requires video analysis)

**Acquisition Strategy**:
- **Primary**: Visual observation from replays
- **Method**: Human observation (action quality assessment)
- **Extract**: Smooth vs laboring action, acceleration quality, fitness signs

**Source Specifics**:
```
Running Style Observation Categories:

1. Action Quality:
   - Smooth Action: Horse traveling effortlessly (good stride, balanced)
   - Laboring Action: Horse struggling (poor stride, unbalanced, working hard)

   Visual Cues:
   - Smooth: Jockey sitting still, horse relaxed, fluid stride
   - Laboring: Jockey pushing/niggling, horse's head bobbing, irregular stride

   Example Smooth Action:
   ```
   Replay Observation:
   - Horse A mid-pack, jockey sitting quietly with hands still
   - Horse has fluid, rhythmic stride (balanced action)
   - Ears pricked forward (alert, comfortable)
   - Assessment: SMOOTH ACTION (high quality, good fitness)
   - Prediction Impact: +5-8% boost (good action indicates fitness)
   ```

   Example Laboring Action:
   ```
   Replay Observation:
   - Horse B mid-pack, jockey niggling constantly (pushing with hands)
   - Horse has irregular stride (head bobbing, unbalanced)
   - Laboring to keep up (struggling to maintain position)
   - Assessment: LABORING ACTION (poor quality, fitness concern)
   - Prediction Impact: -8-12% penalty (poor action indicates issue)
   ```

2. Acceleration Quality:
   - Quick Acceleration: Horse responds immediately to rider's urging (good engine)
   - Slow Acceleration: Horse slow to respond (weak engine, fitness issue)

   Visual Cues:
   - Quick: Jockey asks, horse immediately quickens (instant response)
   - Slow: Jockey asks, horse takes time to respond (sluggish)

   Example Quick Acceleration:
   ```
   Replay Observation (400m mark):
   - Jockey sits down and pushes (asks horse to quicken)
   - Horse responds IMMEDIATELY (surges forward within 2-3 strides)
   - Passes 2 rivals quickly (good acceleration)
   - Assessment: QUICK ACCELERATION (good engine)
   - Prediction Impact: +5-8% boost (quality acceleration)
   ```

3. Fitness Indicators:
   - Ears Pricked: Horse alert and comfortable (good fitness)
   - Ears Back: Horse working hard or distressed (fitness concern)
   - Breathing: Normal breathing vs labored breathing (fitness indicator)
   - Pulling Up: How horse pulls up after race (fitness recovery)

   Visual Cues:
   - Fit: Ears pricked throughout, pulls up well, barely blowing
   - Unfit: Ears back, labored breathing, pulls up distressed, blowing hard

   Example Fit Horse:
   ```
   Replay Observation (entire race + after finish):
   - Horse C traveled with ears pricked entire race (alert, comfortable)
   - Finished race strongly (good finishing effort)
   - Pulled up quickly after finish (good control)
   - Barely blowing 30 seconds after race (excellent fitness)
   - Assessment: EXCELLENT FITNESS (all indicators positive)
   - Prediction Impact: +5-8% boost (fitness confirmed)
   ```

4. Tactical Pattern Quality:
   - Good Position: Horse in ideal position for running style (leader in front, closer with cover)
   - Poor Position: Horse in bad position (leader trapped, closer stuck wide)

   Visual Cues:
   - Good: Horse where jockey wants it (tactical plan executed)
   - Poor: Horse fighting for position or forced wide (tactical plan failed)

   Example Good Tactical Position:
   ```
   Replay Observation:
   - Horse D (closer) settled mid-pack with cover (ideal for closer)
   - Saved ground on rail entire race (shortest trip)
   - Peeled out at 300m with clear run (perfect timing)
   - Assessment: EXCELLENT TACTICAL RIDE (plan executed perfectly)
   - Prediction Impact: +3-5% boost (tactical advantage realized)

### 15.3 Incident & Interference Review (Bad Luck Identification)

**Importance**: 9/10 (Identifying legitimate excuses)

**Prediction Power**: 10-15% (Severe interference completely excuses poor result)

**Data Availability**: Medium (Stewards reports + replay observation)

**Acquisition Strategy**:
- **Primary**: Stewards reports (11.1) - official incident documentation
- **Secondary**: Visual replay confirmation (human observation)
- **Extract**: Checks, interference, wide runs, blocked runs

**Source Specifics**:
```
Incident Review Data Sources:

1. Stewards Reports (Primary - Category 11.1):
   - URL: https://www.racingnsw.com.au / https://www.racing.com
   - Data: Official incident reports (checks, interference, inquiry results)
   - Quality: AUTHORITATIVE (official regulatory documentation)
   - Coverage: 100% (all significant incidents documented)
   - Method: Web scraping (already covered in Part 3d)

2. Visual Replay Confirmation (Secondary):
   - Method: Watch replay to assess severity of incident
   - Purpose: Verify stewards report + assess impact severity
   - Time: 1-2 minutes (watch incident moment only)

Incident Categories & Impact:

1. SEVERE Interference (Checked Heavily):
   - Description: Horse checked heavily, lost all momentum, impossible to recover
   - Stewards Report: "Checked heavily at 300m, lost all chance"
   - Visual Confirmation: Horse pulled back sharply, gap closed, fell back several lengths
   - Impact: COMPLETELY FORGIVE POOR RESULT (+15-20% boost)
   - Reasoning: Result entirely due to bad luck, not horse's ability

   Example:
   ```
   Stewards Report: "Horse X checked heavily at 300m when runner in front shifted out. Lost all chance."
   Replay Observation: Horse X traveling 2nd, positioned well. At 300m, leader shifted out suddenly (no room). Jockey had to check hard (pull reins sharply). Horse X lost 4+ lengths in 50m, fell to 8th.
   Assessment: SEVERE INTERFERENCE (completely forgive poor result)
   Prediction Impact: +18-22% boost (totally forgive 8th place, treat as if never happened)
   ```

2. MODERATE Interference (Checked, Some Impact):
   - Description: Horse checked, lost momentum briefly, recovered somewhat
   - Stewards Report: "Checked at 400m"
   - Visual Confirmation: Horse steadied, lost 1-2 lengths, continued racing
   - Impact: PARTIALLY FORGIVE RESULT (+8-12% boost)
   - Reasoning: Bad luck affected result, but not total disaster

   Example:
   ```
   Stewards Report: "Horse Y checked at 400m."
   Replay Observation: Horse Y traveling 4th. At 400m, gap tightened slightly. Jockey steadied briefly, lost 1-2 lengths, continued.
   Assessment: MODERATE INTERFERENCE (some bad luck, but not severe)
   Prediction Impact: +10-12% boost (partially forgive result, had some impact)
   ```

3. MINOR Interference (Slight Check):
   - Description: Horse slightly interfered with, minimal impact
   - Stewards Report: "Steadied at 500m" (minor mention)
   - Visual Confirmation: Very brief check, minimal impact
   - Impact: SMALL ADJUSTMENT (+3-5% boost)
   - Reasoning: Minor bad luck, but not significant

4. Wide Run (No Cover):
   - Description: Horse raced wide entire race (extra distance, no cover)
   - Stewards Report: May not be mentioned (not "interference" per se)
   - Visual Confirmation: Horse three-wide or four-wide, extra ground covered
   - Impact: EXCUSE POOR RESULT (+8-12% boost)
   - Reasoning: Extra distance covered = harder run, performance better than result

   Example:
   ```
   Replay Observation:
   - Horse Z drawn wide barrier (11)
   - Forced to settle three-wide with no cover (entire race)
   - Extra ground covered: Estimated 8-12 meters (significant)
   - Finished 5th (moderate result on paper)
   - Assessment: TOUGH RUN (wide entire race, extra distance)
   - Prediction Impact: +10-12% boost (performance better than result, draw cost horse)
   ```

5. Blocked for Run (No Clear Running):
   - Description: Horse had no clear run in straight (boxed in, had to check/wait)
   - Stewards Report: "Held up for clear run from 300m to 200m"
   - Visual Confirmation: Horse hemmed in, jockey pulling/checking, finally got clear late
   - Impact: EXCUSE POOR RESULT (+10-15% boost)
   - Reasoning: Had to wait for room, lost crucial momentum at key moment

   Example:
   ```
   Stewards Report: "Horse A held up for clear run from 300m to 200m."
   Replay Observation: Horse A positioned 5th turning for home. Runner in front and outside blocked path. Jockey pulling hard (trying to find gap). Finally got clear at 200m but too late. Finished 6th.
   Assessment: BLOCKED FOR RUN (legitimate excuse, no clear running)
   - Prediction Impact: +12-15% boost (was traveling well, blocked at crucial moment)
