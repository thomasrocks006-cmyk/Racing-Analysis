### 16.1 Health Status Beyond Public Knowledge (Hidden Health Issues)

**Importance**: 10/10 (Hidden health issues critically impact performance)

**Prediction Power**: 15-25% (Undisclosed health problems can completely compromise performance)

**Data Availability**: Very Hard (Insider knowledge, mostly unavailable)

**Acquisition Strategy**:
- **Primary**: Veterinary reports from stewards (11.6) - official health disclosures
- **Secondary**: Stable whispers (social media, insider tips - 13.5)
- **Tertiary**: Trackwork/trial behavior analysis (laboring, off-stride - 10.4)
- **Inference**: Detect health issues from pattern breaks (form drop, behavior change)

**Source Specifics**:
```
Health Status Data Sources:

1. Official Veterinary Reports (Primary - Category 11.6):
   - Source: Stewards reports (Racing NSW/VIC)
   - Data: Official health disclosures (bleeding, lame, pulled up sore, stand-down periods)
   - Coverage: 30-40% (only REPORTED issues, many undisclosed)
   - Quality: AUTHORITATIVE when available
   - **Note**: Already covered in Part 3d (Category 11.6)

2. Stable Whispers (Secondary - Category 13.5):
   - Source: Social media (Twitter/X), racing forums, insider accounts
   - Data: Rumored health issues ("heard horse not 100%", "stable concerned")
   - Coverage: 10-20% (rare genuine insider information)
   - Quality: VARIABLE (genuine insiders vs noise)
   - **Note**: Already covered in Part 4b (Category 13.5)

3. Trackwork/Trial Behavior Inference (Tertiary - Category 10.4):
   - Source: Barrier trial behavior observations
   - Indicators: "Pulled up distressed", "laboring", "off-stride", "blowing hard"
   - Coverage: 15-25% (trials don't always report detailed health observations)
   - Quality: INDIRECT (behavior indicates possible health issue)
   - **Note**: Already covered in Part 3c (Category 10.4)

4. Pattern Break Inference (Inference Method):
   - Method: Detect SUDDEN form drops + behavior changes = possible health issue
   - Indicators:
     * Sudden form drop (from winning to unplaced with no obvious excuse)
     * Behavioral change (aggressive ‚Üí timid, or vice versa)
     * Repeated poor performances (3+ bad runs = persistent issue)
     * Stewards post-race comment: "Pulled up lame/sore" (undisclosed before race)

Health Issue Categories:

1. DISCLOSED Health Issues (Official):
   - Bleeding (Category 11.6): Mandatory 3-month stand-down (KNOWN)
   - Post-race lameness (Category 11.6): "Pulled up lame" (DISCLOSED after race)
   - Veterinary certificate required (Category 11.6): Stand-down period (DISCLOSED)

   Application:
   - IF Disclosed health issue: -20-30% win probability penalty (confirmed problem)
   - IF Recently returned from health stand-down: -10-15% penalty first run back (fitness concern)

2. SUSPECTED Health Issues (Inferred):
   - Sudden form drop with no excuse (3 wins ‚Üí 3 unplaced runs)
   - Post-race comments "pulled up distressed" (undisclosed issue)
   - Poor trial behavior "laboring" + poor race result (health suspected)

   Application:
   - IF Suspected health issue (pattern break): -15-20% win probability penalty (probable problem)

3. INSIDER-REPORTED Health Issues (Rumored):
   - Social media whisper: "Horse not 100%, stable concerned"
   - Forum post: "Heard Horse X has minor injury, racing underdone"

   Application:
   - IF Credible insider reports health concern: -12-18% win probability penalty (if insider credible)
   - IF Non-credible source: Ignore (noise)

Hidden Health Issue Detection Framework:

Pattern 1: Sudden Form Drop (No Obvious Excuse)

Example:
```
Horse X Recent Form:
- 4 starts ago: 1st (won easily)
- 3 starts ago: 1st (won again)
- 2 starts ago: 2nd (close second)
- Last start: 10th of 12 (unplaced, beaten 12 lengths)
- Stewards report last start: No interference, no excuse mentioned
- Post-race comment: "Horse pulled up sore, had veterinary examination"

Analysis:
- Sudden form drop (winning ‚Üí 10th) with NO OBVIOUS EXCUSE
- Post-race comment reveals health issue (pulled up sore)
- Health issue was UNDISCLOSED before race (hidden)

Prediction Impact Today:
- IF No veterinary clearance shown: -20-25% win probability penalty (health concern unresolved)
- IF Veterinary clearance + trial since: -10-15% penalty (health issue treated, but caution)
- IF No trial since last run: -15-20% penalty (health status unknown, high risk)
```

Pattern 2: Repeated Poor Performances (Persistent Issue)

Example:
```
Horse Y Recent Form:
- Last 3 starts: 8th, 9th, 7th (all unplaced)
- Previous form before that: 1st, 2nd, 1st (was winning)
- No obvious excuses (no interference, reasonable draws, same jockey)
- No disclosed health issues (no stewards reports of injury)

Analysis:
- Persistent poor form (3 bad runs in row) with no explanation
- Pattern suggests UNDERLYING ISSUE (health, behavior, or lost ability)
- Issue is HIDDEN (not disclosed publicly)

Prediction Impact Today:
- IF 4th run with no improvement signals: -15-20% win probability penalty (persistent issue)
- IF Trial since showing improvement: -5-10% penalty (possible recovery)
- IF No trial + no explanation: -18-25% penalty (unknown issue, high risk)
```

Pattern 3: Post-Race Distress (Undisclosed Before Next Race)

Example:
```
Horse Z Last Start:
- Finished 5th (moderate result)
- Post-race comment: "Horse pulled up distressed, breathing heavily"
- Stewards: No veterinary examination ordered (minor concern)
- Today: Racing again 3 weeks later (quick backup)
- No trial between runs (no fitness test)

Analysis:
- Post-race distress indicates POSSIBLE HEALTH ISSUE
- No trial since (health status UNKNOWN)
- Quick backup (3 weeks) may not be enough recovery

Prediction Impact Today:
- -12-18% win probability penalty (undisclosed health concern + no trial = risky)
```

Health Status Integration:

Priority Hierarchy:
1. Official veterinary reports (authoritative, -20-30% if active issue)
2. Post-race distress + no trial since (-12-18% penalty)
3. Sudden form drop + no excuse (-15-20% penalty)
4. Credible insider whispers (-12-18% penalty if credible)
5. Persistent poor form (-15-20% penalty)

Confidence Levels:
- Official disclosure: VERY HIGH confidence (authoritative)
- Pattern inference: MODERATE confidence (indirect evidence)
- Insider whispers: LOW-MODERATE confidence (depends on source credibility)
```

**Missing Variables**:
- **Insider Access**: True insider information rarely available (stable confidentiality)
- **Undisclosed Issues**: Most minor health issues never disclosed (competitive advantage for stable)
- **Inference Ambiguity**: Pattern breaks may be health OR ability loss OR behavioral (hard to distinguish)
- **No Confirmation**: Cannot confirm suspected health issues (guessing based on patterns)
- **False Positives**: Some form drops are natural variance (not health issues)

**Implementation Priority**: **Phase 2** (Critical but data-limited, use inference methods)

**Notes**:
- Hidden health issues are CRITICAL but RARELY DISCLOSED (competitive secrecy)
- Official veterinary reports (11.6) cover ~30-40% of health issues (disclosed only)
- ~60-70% of health issues are UNDISCLOSED (hidden competitive advantage)
- Must INFER health issues from pattern breaks (sudden form drop + no excuse)
- Post-race distress ("pulled up sore") + no trial since = -12-18% penalty (health unknown)
- Persistent poor form (3+ bad runs) = -15-20% penalty (underlying issue suspected)
- Credible insider whispers valuable but rare (verify source credibility)
- ChatGPT has MINIMAL health status analysis (~5%, only obvious disclosed issues)
- **üö® COMPETITIVE EDGE**: Systematic pattern break detection + inference framework (ChatGPT 5% ‚Üí Us 50% via inference)

### 16.2 Fatigue & Recovery Status (Campaign Fatigue, Freshness)

**Importance**: 9/10 (Fatigue significantly degrades performance)

**Prediction Power**: 12-20% (Late campaign fatigue common)

**Data Availability**: Medium (Infer from starts/breaks pattern)

**Acquisition Strategy**:
- **Primary**: Calculate campaign length (starts since spell)
- **Secondary**: Time since last start analysis
- **Tertiary**: Post-race fatigue indicators (pulled up distressed)
- **Pattern**: Detect late campaign decline

**Source Specifics**:
```
Fatigue & Recovery Data Sources:

1. Campaign Length Calculation (Primary):
   - Source: Horse's race history (number of starts since spell)
   - Data: Count starts from last spell (8+ weeks break)
   - Method: Calculate from racing.com form guide
   - Indicators:
     * Fresh (1st-3rd run from spell): Peak freshness
     * Mid-campaign (4th-6th run): Peak fitness
     * Late campaign (7th-10th run): Fatigue risk
     * Very late campaign (11+ runs): HIGH fatigue risk

2. Time Since Last Start Analysis (Secondary):
   - Source: Days between races (racing.com form guide)
   - Indicators:
     * 7-14 days: Normal spacing (adequate recovery)
     * 15-21 days: Comfortable spacing (good recovery)
     * <7 days: Quick backup (fatigue risk)
     * 22-28 days: Longer break (freshening up)
     * 28-56 days: Short spell (refresh)
     * 56+ days: Spell (returning fresh)

3. Post-Race Fatigue Indicators (Tertiary):
   - Source: Stewards reports (11.2) - post-race jockey comments
   - Indicators:
     * "Pulled up distressed" (severe fatigue)
     * "Breathing heavily" (fatigue)
     * "Pulled up well" (good recovery, not fatigued)

Campaign Fatigue Framework:

Campaign Stage Analysis:

Stage 1: FRESH (1st-3rd Run from Spell)
```
Characteristics:
- Horse returning from 8+ weeks spell (rested, refreshed)
- First few runs back (building fitness)
- Often improving with each run (fitness building)

Fatigue Risk: LOW (freshness advantage)

Performance Pattern:
- 1st run: -5-10% (fitness building, may need run)
- 2nd run: +3-5% (fitness improved, "second-up" angle)
- 3rd run: +5-8% (peak fitness approaching)

Application:
- 1st run from spell: -5-10% penalty if no trial (fitness concern) OR +3-5% if strong trial (ready)
- 2nd run from spell: +5-8% boost ("second-up" advantage, fitness improved)
- 3rd run from spell: +5-8% boost (approaching peak fitness)
```

Stage 2: MID-CAMPAIGN (4th-6th Run from Spell)
```
Characteristics:
- Horse in campaign rhythm (racing regularly)
- Peak fitness period (fittest stage of campaign)
- Consistent form expected

Fatigue Risk: LOW-MODERATE (peak fitness, but fatigue starting)

Performance Pattern:
- 4th-5th run: +8-12% (PEAK FITNESS, ideal stage)
- 6th run: +3-5% (still good fitness, fatigue beginning)

Application:
- 4th-5th run: +8-12% boost (PEAK FITNESS ADVANTAGE, best form expected)
- 6th run: +3-5% boost (still fit, but fatigue starting to factor)
```

Stage 3: LATE CAMPAIGN (7th-10th Run from Spell)
```
Characteristics:
- Horse racing for extended period (2-4 months since spell)
- Fatigue accumulating (wear and tear)
- Form often declining (tired horses)
- Recovery between runs slower

Fatigue Risk: MODERATE-HIGH (fatigue affecting performance)

Performance Pattern:
- 7th-8th run: -5-10% (fatigue factor, performance declining)
- 9th-10th run: -10-15% (HIGH fatigue, form typically poor)

Application:
- 7th-8th run: -8-12% penalty (fatigue factor, declining form)
- 9th-10th run: -12-18% penalty (HIGH fatigue risk, needs spell)

Exception:
- IF Horse maintaining strong form at 7th-8th run: Reduce penalty to -3-5% (tougher horse, handling campaign)
```

Stage 4: VERY LATE CAMPAIGN (11+ Runs from Spell)
```
Characteristics:
- Horse racing for 4+ months without break
- Extreme fatigue risk (very tired)
- Form almost always poor (exhausted horses)

Fatigue Risk: VERY HIGH (extreme fatigue, needs spell urgently)

Performance Pattern:
- 11+ runs: -15-25% (VERY HIGH fatigue, performance severely compromised)

Application:
- 11+ runs: -18-25% penalty (EXTREME fatigue risk, avoid betting)

Exception:
- RARE: Elite iron horses (1-2% of horses) handle long campaigns (famous stayers like Makybe Diva)
- IF Elite iron horse with strong form: Reduce penalty to -5-8% (exceptional toughness)
```

Time Since Last Start Analysis:

Quick Backup (< 7 Days):
```
Scenario: Horse racing <7 days since last start (e.g., Wednesday ‚Üí Saturday)

Fatigue Risk: HIGH (insufficient recovery)

Performance Impact:
- -10-15% win probability penalty (quick backup, fatigue risk)

Exception:
- IF Horse known "quick backup specialist" (proven record <7 days): Reduce to -3-5% penalty
- IF Last start very easy run (coasted, not fully extended): Reduce to -5-8% penalty
```

Normal Spacing (7-21 Days):
```
Scenario: Horse racing 7-21 days since last start (normal pattern)

Fatigue Risk: LOW (adequate recovery)

Performance Impact:
- Neutral (¬±0%) - normal spacing, no fatigue concern
```

Freshening Break (22-56 Days):
```
Scenario: Horse racing 22-56 days since last start (short spell/freshen up)

Fatigue Risk: LOW (refreshed)

Performance Impact:
- +5-10% win probability boost (freshened up, revitalized)

Note: Different from "first-up" (56+ days spell) - this is mid-campaign refresh
```

Spell Returning (56+ Days):
```
Scenario: Horse returning from 56+ days spell (full break)

Fatigue Risk: LOW (fully rested)

Performance Impact:
- See Stage 1 (Fresh, 1st-3rd run from spell)
```

Fatigue Indicator Integration:

Combining Campaign Length + Time Since Last Start:

Scenario 1: Late Campaign (8th run) + Quick Backup (6 days)
```
- Campaign fatigue: -8-12% penalty (late campaign)
- Quick backup: -10-15% penalty (insufficient recovery)
- Combined: -18-27% penalty (VERY HIGH RISK, multiple fatigue factors)
```

Scenario 2: Mid-Campaign (5th run) + Normal Spacing (14 days)
```
- Campaign fitness: +8-12% boost (peak fitness)
- Normal spacing: Neutral (¬±0%)
- Combined: +8-12% boost (IDEAL, peak fitness + adequate recovery)
```

Scenario 3: Late Campaign (9th run) + Freshening Break (28 days)
```
- Campaign fatigue: -10-15% penalty (late campaign)
- Freshening break: +5-10% boost (refreshed)
- Combined: -5-8% penalty NET (fatigue partially offset by freshen up)
```

Post-Race Fatigue Indicators:

Stewards Post-Race Comments (Category 11.2):
```
"Horse pulled up distressed, breathing heavily"
- Interpretation: SEVERE FATIGUE (horse exhausted)
- Application Today: -15-20% penalty if racing back within 2 weeks (fatigue unresolved)

"Horse pulled up well, barely blowing"
- Interpretation: GOOD RECOVERY (not fatigued)
- Application Today: +5-8% boost (good fitness, handles campaign well)
```
```

**Missing Variables**:
- **Individual Recovery Rates**: Some horses recover faster than others (varies by individual)
- **Workout Intensity**: Cannot measure training intensity between races (affects fatigue)
- **Age Factor**: Younger horses (3-4YO) typically recover faster than older horses (7YO+)
- **Track Conditions Impact**: Heavy tracks cause more fatigue than Good tracks (cannot quantify exactly)
- **Race Intensity**: Some races more taxing than others (fast pace vs slow pace)

**Implementation Priority**: **Phase 2** (CRITICAL, data easily available via calculation)

**Notes**:
- Campaign fatigue is REAL and QUANTIFIABLE (late campaign = -10-15% penalty)
- Peak fitness: 4th-6th run from spell (+8-12% boost)
- Late campaign: 7th-10th run (-8-15% penalty)
- Very late campaign: 11+ runs (-18-25% penalty, extreme fatigue)
- Quick backup (<7 days) = -10-15% penalty (insufficient recovery)
- Freshening break (22-56 days) = +5-10% boost (refreshed)
- Must combine campaign length + time since last start (multiple factors)
- Post-race fatigue indicators ("pulled up distressed") = -15-20% penalty if quick backup
- ChatGPT may mention "first-up" qualitatively (~40%) but no systematic campaign fatigue analysis
- **üö® COMPETITIVE EDGE**: Systematic campaign fatigue quantification (ChatGPT 40% ‚Üí Us 85%)

### 16.3 Travel & Transportation Fatigue (Interstate/Regional Racing)

**Importance**: 7/10 (Travel affects performance, especially long distances)

**Prediction Power**: 8-15% (Interstate travel particularly impactful)

**Data Availability**: Easy (Infer from stable location vs race location)

**Acquisition Strategy**:
- **Primary**: Compare trainer location vs race location
- **Calculate**: Travel distance and impact
- **Identify**: Interstate raiders (horses traveling from different state)

**Source Specifics**:
```
Travel Fatigue Data Sources:

1. Trainer Location (Primary):
   - Source: Racing.com form guide (trainer details)
   - Data: Trainer's primary stable location (e.g., "Chris Waller - Sydney")
   - Method: Extract trainer location, compare to race venue

2. Race Venue (Primary):
   - Source: Race information (venue)
   - Data: Race location (e.g., "Flemington, VIC" or "Royal Randwick, NSW")

3. Travel Distance Calculation:
   - Method: Calculate distance between trainer location and race venue
   - Categories:
     * Local (0-50km): No travel fatigue
     * Regional (50-200km): Minimal travel fatigue
     * Interstate (200-1000km): Moderate-significant travel fatigue
     * Long Interstate (1000+ km): Significant travel fatigue

Travel Fatigue Framework:

Travel Distance Categories:

1. LOCAL Racing (0-50km):
```
Example: Trainer based at Flemington, racing at Flemington
Distance: 0-10km (local)

Travel Fatigue: NONE (no travel impact)

Performance Impact:
- Neutral (¬±0%) - no travel disadvantage
- May have HOME TRACK ADVANTAGE (+3-5% if trainer track specialist)
```

2. REGIONAL Racing (50-200km):
```
Example: Trainer based in Ballarat, racing at Flemington
Distance: 110km (regional)

Travel Fatigue: MINIMAL (short drive, same day travel)

Performance Impact:
- -3-5% win probability penalty (minor travel fatigue, minimal impact)
```

3. INTERSTATE Racing (200-1000km):
```
Example: Trainer based in Sydney (Randwick), racing at Flemington (Melbourne)
Distance: 870km (interstate, flew or trucked)

Travel Method:
- Flight: 1.5 hour flight (faster but stressful for horses)
- Truck: 8-10 hour drive (slower but less stressful)

Travel Fatigue: MODERATE-SIGNIFICANT (interstate travel stressful)

Performance Impact:
- First-time interstate: -12-18% penalty (unfamiliar, stressful travel)
- Proven interstate performer: -5-8% penalty (handles travel better)
- Elite trainer known for interstate raids: -3-5% penalty (expert preparation)

Exception:
- IF Horse has strong interstate record (won multiple times interstate): -3-5% penalty only (proven traveler)
- IF Elite trainer with strong interstate record (e.g., Chris Waller Sydney ‚Üí Melbourne): -5-8% penalty (specialist preparation)
```

4. LONG INTERSTATE Racing (1000+ km):
```
Example: Trainer based in Perth, racing at Flemington (Melbourne)
Distance: 2700km (very long interstate, always flown)

Travel Fatigue: SIGNIFICANT (very long travel, high stress)

Performance Impact:
- -15-25% win probability penalty (very long travel, major disadvantage)

Exception:
- IF Horse has proven record at this distance (won at this venue before): -8-12% penalty (proven traveler)
- RARE: Elite Perth trainers targeting eastern races (well-prepared): -10-15% penalty
```

Interstate Raider Analysis:

Interstate Raider Pattern:

Characteristics:
- Trainer based in different state than race venue
- Horse trucked or flown to race (travel stress)
- Unfamiliar track (additional challenge)
- Typical pattern: Target major races (Group races, valuable races)

Performance Statistics:
- Interstate raiders: ~8-12% win rate (vs ~15-18% local horses)
- Disadvantage: -40-50% relative win rate (vs locals)

Application:
```
Scenario 1: Sydney Horse Racing in Melbourne (Interstate)

- Race: Flemington, Melbourne
- Travel: 870km (flight or truck)
- First-time at Flemington: -15-20% penalty (travel + unfamiliar track)

Scenario 2: Melbourne Horse Racing in Melbourne (Local)
- Trainer: Based Melbourne (Flemington)

```
Elite Interstate Trainers:

Some trainers SPECIALIZE in interstate raids:

IF Elite interstate trainer:
  THEN Reduce travel penalty by 50% (specialist preparation offsets travel)

Example:
```
Chris Waller (Sydney) horse racing at Flemington:
- Standard interstate penalty: -12-18%
- Elite trainer discount: 50% reduction
- Adjusted penalty: -6-9% (still disadvantage, but less severe)
```

Travel Timing:

Timing of Travel Matters:

1. Travel Several Days Before Race (3-7 days early):
   - Horse has time to acclimatize (settle in, recover from travel)
   - Better outcome than arriving day before
   - Penalty: Reduce by 30-40% (e.g., -12% ‚Üí -7%)

2. Travel Day Before Race (1-2 days):
   - Horse has minimal time to recover
   - Still stressed from travel on race day
   - Penalty: FULL penalty (e.g., -12-18%)

Note: Cannot typically determine exact travel timing (not publicly disclosed)
Assume: Standard travel (1-2 days before) unless evidence of early arrival

Travel Fatigue Integration:

Combining Travel + Campaign Fatigue:

Scenario: Interstate Raider (Sydney ‚Üí Melbourne) + Late Campaign (9th run)
```
- Interstate travel: -12-18% penalty
- Late campaign fatigue: -10-15% penalty
- Combined: -22-33% penalty (VERY HIGH RISK, multiple disadvantages)
```

Scenario: Local Horse + Mid-Campaign (5th run)
```
- Local (no travel): +3-5% boost (home advantage)
- Mid-campaign fitness: +8-12% boost (peak fitness)
- Combined: +11-17% boost (IDEAL, multiple advantages)
```
```

**Missing Variables**:
- **Travel Timing**: Cannot determine when horse traveled (days before race)
- **Travel Method**: Cannot determine if flown or trucked (different stress levels)
- **Acclimatization**: Cannot measure how well horse acclimatized (individual variance)
- **Stable Infrastructure**: Some stables have better travel preparation (not quantifiable)
- **Track Familiarity**: Separate from travel, but compounds disadvantage (hard to separate)

**Implementation Priority**: **Phase 2** (Easy to calculate, significant impact)

**Notes**:
- Interstate travel is REAL DISADVANTAGE (-12-18% penalty typically)
- Local horses have HOME ADVANTAGE (+3-5% boost, no travel + track familiarity)
- Elite interstate trainers (Chris Waller, Gai Waterhouse) reduce penalty by ~50% (-12% ‚Üí -6%)
- Proven interstate performers (won at this track before) = -5-8% penalty only (travel experience)
- Long interstate (Perth ‚Üí Melbourne) = -15-25% penalty (very long travel)
- Must combine with other factors (campaign fatigue, track familiarity)
- ChatGPT may mention "interstate" qualitatively (~50%) but no systematic quantification
- **üö® COMPETITIVE EDGE**: Quantified travel impact by distance + trainer specialization (ChatGPT 50% ‚Üí Us 80%)

### 16.4 Age & Career Stage Considerations (Peak Age, Decline)

**Importance**: 7/10 (Age affects performance trajectory)

**Prediction Power**: 5-10% (Peak age advantage, decline penalty)

**Data Availability**: Easy (Horse age in form guide)

**Acquisition Strategy**:
- **Primary**: Extract horse age from form guide
- **Identify**: Peak age horses (4-5YO) vs declining (7YO+)
- **Pattern**: Age-based performance expectations

**Source Specifics**:
```
Age & Career Stage Data:

1. Horse Age (Primary):
   - Source: Racing.com form guide (horse details)
   - Data: Age in years (e.g., "4YO" = 4-year-old)
   - Method: Direct extraction from form data

Age-Based Performance Framework:

Age Categories & Performance Expectations:

1. 2YO (Two-Year-Olds):
```
Characteristics:
- Very young horses (first year of racing)
- Still developing physically and mentally
- Short-distance races only (1000-1400m typically)
- Often inconsistent (learning to race)

Performance Expectations:
- Inconsistent form (learning curve)
- Early maturity advantage (horses that develop early dominate)
- Limited race experience (first few starts)

Application:
- 2YO racing: Age not primary factor (all 2YO races, level playing field)
- Early maturity: IF 2YO with multiple wins already: +5-8% boost (early developer)
- Inexperienced: IF 2YO with 0-2 starts: -5-8% penalty (learning, inexperienced)
```

2. 3YO (Three-Year-Olds):
```
Characteristics:
- Developing horses (second year of racing)
- Physical maturity improving (getting stronger)
- Stepping up in distance (1200-2000m)
- Quality emerging (classics targets)

Performance Expectations:
- Improving horses (fitness and strength building)
- Spring 3YO (Sep-Nov): Often well-prepared for spring classics
- Autumn 3YO (Mar-May): Often well-prepared for autumn classics

Application:
- 3YO vs 3YO races: Neutral (level playing field)
- 3YO vs Older (age-restricted): -5-10% penalty (racing against stronger, more mature horses)
- 3YO Autumn (after spell): +3-5% boost (freshened up, developing)
- 3YO Spring (well-prepared): +3-5% boost (targeted for major races)
```

3. 4YO (Four-Year-Olds):
```
Characteristics:
- Mature horses (physical development complete)
- PEAK AGE for most horses (strength + speed optimal)
- Experienced racers (multiple campaigns)
- Handling longer distances (1400-2400m)

Performance Expectations:
- PEAK PERFORMANCE AGE (best form expected)
- Consistent form (mature, experienced)
- Versatile (can handle various distances/conditions)

Application:
- 4YO: +5-8% win probability boost (PEAK AGE ADVANTAGE)
- 4YO vs younger (3YO): +5-8% boost (maturity advantage)
- 4YO vs older (5-6YO): Neutral (similar maturity)
```

4. 5YO (Five-Year-Olds):
```
Characteristics:
- Fully mature horses
- Still in peak age range (slightly past absolute peak)
- Experienced campaigners
- Often specialists emerging (distance/track preferences clear)

Performance Expectations:
- STRONG PERFORMANCE (still in peak range)
- Consistent form (veteran experience)
- Specialist capabilities clear (knows job)
